Follow following steps to modify existing apps or to create new ones

  • Fork the Seranet.apps Github repository on to your own GitHub account
  • Clone it to your local dev environment and open Seranet.Apps project with VisualStudio (or any other editor)
  • This is a pure html/JS application with no server code, therefore you may use any text editor instead of VS
  • Every applet you see in 'Explore' is a self contained plugin located in app/module/pluginModule folder
  • To create a new applet, you can copy and paste a existing plugin and do modifications accordingly
  • For simple apps with just a link + image, the changes you ned to make are trivial, just read the plugin code you will know
  • You can run the project and test it, through any simple HTTP server (or directly on VisualStudio)
  • Once done, commit/push the changes to your Github repository, then make a pull request to the 'ectechno' original
  • Someone will review your changes and deploy to the production soon :) !!!
